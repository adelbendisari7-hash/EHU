# src/components/ui/

shadcn/ui base components. These are the foundational UI primitives for the entire application.

## Installed Components
Core components available (install more as needed):
- `button.tsx` — Button with variants: default, destructive, outline, secondary, ghost, link
- `input.tsx` — Text input
- `label.tsx` — Form label
- `select.tsx` — Dropdown select (Radix UI based)
- `dialog.tsx` — Modal dialog
- `table.tsx` — HTML table with styled thead/tbody/tr/td
- `badge.tsx` — Inline badge/chip with variants
- `tabs.tsx` — Tab navigation (Radix UI based)
- `card.tsx` — Card container with CardHeader, CardContent, CardFooter
- `toast.tsx` / `toaster.tsx` — Toast notifications (Sonner)
- `dropdown-menu.tsx` — Context/dropdown menu (Radix UI based)
- `form.tsx` — React Hook Form integration wrappers
- `checkbox.tsx`, `radio-group.tsx`, `switch.tsx` — Form controls
- `avatar.tsx` — User avatar with fallback initials
- `separator.tsx` — Visual divider
- `skeleton.tsx` — Loading skeleton placeholder

## Rules
- **Do not modify** files in this directory directly — they are auto-generated by shadcn/ui CLI
- To add a new component: `npx shadcn-ui@latest add [component-name]`
- To customize appearance: use the Tailwind CSS variables in `globals.css` under `:root` / `.dark`
- To extend a component: create a wrapper in the feature-specific components folder (e.g., `components/shared/status-badge.tsx` wraps `Badge`)
